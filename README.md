# hello-world-nginx

This is a simple nginx docker setup.

## Usage

### Start the server

```bash
  $ ./docker-run up
```

### Shut down server

```bash
  $ ./docker-run down
```

### Show logs

```bash
  $ ./docker-run logs
```
