# hello-world-nginx

This is a simple nginx docker setup.

## Usage

### Start the server

```bash
  $ ./docker-run up
```

### Shut down server

```bash
  $ ./docker-run down
```

### Show logs

```bash
  $ ./docker-run logs
```


## Motivation

This was done to try out ngrok but you may use it to test a plethora of things!!!
